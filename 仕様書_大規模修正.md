# 足場図面自動生成システム 大規模修正 仕様書

**作成日**: 2026年3月18日
**対象システム**: 足場図面自動生成システム｜MT興業
**リポジトリ**: https://github.com/mtkougyou2021-sudo/ashiba-scaffold-system

---

## 修正概要

足場図面の作図ロジックを実務に即した形に大幅改修する。
主な修正は以下の3点。

---

## 修正1: 壁離れの東西南北個別入力

### 現状
- 「建物離れ（目標値）」が **1つ** しかなく、全方向同じ離れで自動計算される

### 修正内容
- 東西南北の **4面それぞれ** で壁からの離れを個別入力できるようにする
- 入力フィールドを4つに分割する

### 入力例
| 方向 | 壁からの離れ |
|------|-------------|
| 西壁 | 300mm |
| 東壁 | 320mm |
| 北壁 | 290mm |
| 南壁 | 290mm |

### 影響範囲
- サイドバーの「足場設定」セクションのUI変更
- `calculateLayout()` 関数の修正（方向ごとに異なるoffsetを使用）
- `generate()` 関数の修正
- 平面図・各立面図への反映

---

## 修正2: 作業床（作業層）の起点設定

### 現状
- 地上（GL）から支柱スパン（1800mm）ごとに段を積み上げていく固定ロジック
- 住宅等では軒天高さとの関係で、最終段が中途半端な高さ（例: 900mm下り）になる場合がある

### 修正内容
作業床の起点（スタート位置）を **選択式** にする。

#### パターンA: 地上起点（現行方式）
- 地上から1800mm上りで1段目の作業床を設置
- 以降1800mmピッチで上に積み上げ

#### パターンB: 軒天起点（逆算方式）
- **軒天高さから1800mm下り** で最上段の作業床位置を決定
- そこから下方向に1800mmピッチで作業床を配置
- 軒天付近の作業床が作業しやすい高さに自動調整される

### 追加入力項目
| 項目 | 説明 |
|------|------|
| 作業床起点 | 「地上から上り」or「軒天から下り」選択 |
| 作業床段数 | 軒天起点の場合、軒天から何段下ろすか（例: 2段） |
| 1段目高さ | 地上起点の場合の最初の作業床高さ（デフォルト: 1800mm） |

### 計算例（軒天起点の場合）
```
軒天高さ: 7500mm
1段目作業床: 7500 - 1800 = 5700mm
2段目作業床: 5700 - 1800 = 3900mm
3段目作業床: 3900 - 1800 = 2100mm
（以下、GLまで繰り返し）
```

## 修正3: 足場部材の450mmピッチ配置ルール

### 現状
- 手すり・中桟の配置ルールが実務と異なる

### 修正内容
作業床から **450mmピッチ（コブ）** で以下の部材を配置する。

| コブ | 高さ（作業床から） | 部材 |
|------|-------------------|------|
| 1コブ目 | +450mm | **中桟**（なかざん） |
| 2コブ目 | +900mm | **手すり** |
| 3コブ目 | +1200mm | なし（何も配置しない） |
| 4コブ目 | +1800mm | **次の作業床** |

### 図面上の表現
- **作業床**: 太い水平線（布板）として描画
- **中桟（450mm）**: 細い水平線として描画
- **手すり（900mm）**: 手すり記号として描画
- **1200mm位置**: 何も描画しない
- **1800mm位置**: 次の作業床（布板）として描画

### 影響範囲
- `drawElevation()` 内の段描画ロジック全面改修
- `drawFuriban()` → 作業床位置のみに描画するよう変更
- `drawTesuri()` → 作業床+900mmの位置に描画するよう変更
- 中桟描画関数の新規追加（作業床+450mmの位置）
- 段のループ処理を450mmピッチに変更

## 技術的な修正ポイント

1. **`getParams()`** — 東西南北の離れ値、作業床起点方式、段数を取得
2. **`calculateLayout()`** — 方向ごとに異なる離れ値で割付計算
3. **`generate()`** — 作業床位置の計算ロジック変更
4. **`drawElevation()`** — 450mmピッチでの部材描画
5. **新規: `drawNakazan()`** — 中桟の描画関数
6. **`drawFuriban()`** — 作業床位置の判定ロジック追加
7. **`drawTesuri()`** — 手すり位置の判定ロジック変更（作業床+900mm）
